# Soumyadipta Das Portfolio

A GitHub Pages-ready static portfolio for Soumyadipta Das.

## Publish On GitHub Pages

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Settings > Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Choose `main` and `/root`.
6. Save.

GitHub will publish the site at:

```text
https://soumyadipta2020.github.io/soumyadipta-das/
```

## Custom Domain

When you have a domain, add it in `Settings > Pages > Custom domain`.
GitHub will create or update the `CNAME` file. Then point your DNS records to GitHub Pages as instructed by GitHub.

## Local Preview

Open `index.html` directly in a browser. No build step is required.
